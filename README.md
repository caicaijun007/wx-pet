## 微信小程序项目

### 萌宠（pet）

涉及知识点：
- 微信小程序
- ES6
- HTML5
- CSS3
- Iconfont
- php
- mysql



项目特色：
- 组件提取与复用
- 模块化数据
- 公共方法提取
- 响应布局（flex布局、rpx单位）
- iconfont图标
- CSS3样式处理
- 用户体验细节处理



项目难点：
- 重构与简化代码
- 缓存处理
- 细节处理
- 优化体验
